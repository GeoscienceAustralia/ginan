◆ nominalYawGalIov()

double nominalYawGalIov ( Vector3d &  eSunOrf)

Calculate nominal (ideal) sat yaw for GAL-IOV sats.

Follows original GAL convention. Roughly equal to nominalYawGps() + PI Ref: https://www.gsc-europa.eu/support-to-developers/galileo-satellite-metadata#3.1 Note: Reference equation is atan2(-x,-y), but this == nominalYawGps(). Conversely, atan2(x,y) gives the desired value of nominalYawGps() + PI/2 so is used below.

Parameters
eSunOrfUnit vector to Sun in Orbital reference frame {A,-C,-R}
